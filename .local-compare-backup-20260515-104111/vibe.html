与AI协作构建的工具与项目

字幕处理自动化

Python · 2026

从字幕清洗到文档导出，一次命令完成多步流程。这个项目起源于一个简单的厌倦：每次处理字幕文件都要重复同样的步骤。与 AI 合作，把这个流程压缩成了一行命令。

目前支持 .srt 格式的去重、时间轴校正、中英法三语对照导出。最意外的收获是在这个过程里学会了如何用自然语言描述一个技术问题。

个人站自动部署

Bash · Nginx · 2026

本地修改后一行命令自动备份、上传、检验 Nginx 配置并重载服务。部署应该是无感的基础设施，而不是每次都需要谨慎对待的仪式。

这个脚本现在管理着这个网站的所有 HTML 文件的同步。它在每次部署前创建带时间戳的备份，失败时自动回滚，运行结果简洁到只输出一行。

法语学习工具页

HTML · JavaScript · 2026

把词汇卡片、表达模板与复习计划集中管理。学语言最大的障碍不是词汇量，而是遗忘的速度总是快于积累的速度。这个工具试图解决这个问题。

用纯 HTML + JS 构建，无框架依赖，可以直接在浏览器里运行。数据存在 localStorage 里，离线也能用。整个项目从想法到可用，花了不到两小时。

翻译工作流脚本

Python · 持续迭代

批量文档处理与格式一致化，减少重复劳动。翻译工作里有大量机械性的格式工作：对齐段落、统一标点、清理多余空格。这些都不需要人来做。

脚本现在可以处理 .docx、.txt 和 .srt 三种格式，输出统一的双语对照文档。它不替代翻译，它只负责翻译之外的所有繁琐。
